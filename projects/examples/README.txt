A# CMD Development Quick Start 
================================ 
 
1. Create new project: 
   cmd_new my_project 
 
2. Edit your code: 
   cmd_edit cmd_projects\my_project.ash 
 
3. Compile and run: 
   cmd_compile cmd_projects\my_project.ash 
 
4. For AI projects: 
   cmd_ai my_ai_project 
 
Available commands: 
- cmd_help          Show help 
- cmd_new name      Create basic project 
- cmd_ai name       Create AI project 
- cmd_edit file     Edit A# file 
- cmd_compile file  Compile and run
